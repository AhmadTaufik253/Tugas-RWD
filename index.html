HTML (Hypertext Markup Language)

      HTML (Hypertext Markup Language) adalah sebuah bahasa markup yang digunakan untuk membuat dan mengatur konten pada halaman web. HTML terdiri dari serangkaian tag atau kode yang digunakan untuk memberikan informasi tentang struktur dan tampilan konten pada sebuah halaman web.
HTML digunakan untuk membuat dokumen web yang terdiri dari teks, gambar, video, audio, dan elemen interaktif lainnya. Dokumen HTML terdiri dari tag atau kode HTML yang diapit oleh tanda kurung sudut (< >). Setiap tag HTML memiliki fungsi tertentu untuk menentukan tampilan dan fungsi dari elemen pada halaman web.
Contoh tag HTML sederhana adalah tag <p> untuk menandai sebuah paragraf teks, dan tag <img> untuk menampilkan sebuah gambar. Selain itu, HTML juga mendukung penggunaan CSS (Cascading Style Sheets) untuk mengatur tampilan dari elemen pada halaman web, dan JavaScript untuk membuat interaksi dan animasi yang dinamis pada halaman web.
HTML sangat penting dalam pengembangan web karena ini adalah dasar dari semua halaman web. Dengan memahami HTML, Anda dapat membuat konten web yang terstruktur dan mudah dimengerti oleh mesin pencari, mempercepat waktu muat halaman, dan meningkatkan pengalaman pengguna.

CSS (Cascading Style Sheets)

      CSS (Cascading Style Sheets) adalah bahasa pemrograman yang digunakan untuk mengatur tampilan atau styling halaman web. CSS memungkinkan pengembang web untuk memisahkan antara konten atau struktur halaman web dengan tampilan atau desain halaman web.
Dengan CSS, pengembang web dapat menentukan warna, jenis huruf, ukuran, jarak antar elemen, posisi, dan efek visual lainnya pada elemen HTML. CSS juga mendukung pembuatan layout halaman web yang kompleks dan responsif.
CSS dibuat sebagai standar web oleh W3C (World Wide Web Consortium) dan sekarang sudah mencapai versi CSS3. Dalam pengembangan web, CSS dapat digunakan secara internal, internal inline, atau eksternal dalam file terpisah.
CSS juga mendukung konsep cascading (bertingkat), artinya beberapa aturan styling dapat diterapkan pada elemen yang sama dan hasilnya akan ditentukan berdasarkan prioritas dan tingkat spesifisitas aturan tersebut.
CSS digunakan secara luas dalam pengembangan web modern, baik untuk mengatur tampilan halaman web statis maupun dinamis. Kombinasi antara HTML, CSS, dan JavaScript menjadi fondasi utama dalam pengembangan aplikasi web modern.

JavaScript

      JavaScript adalah bahasa pemrograman scripting yang sering digunakan untuk pengembangan aplikasi web. JavaScript awalnya dibuat untuk membuat halaman web menjadi lebih interaktif dan responsif dengan mengontrol perilaku dari elemen HTML.
JavaScript dapat digunakan untuk membuat efek animasi, validasi form, mengubah tampilan halaman web secara dinamis, mengirim dan menerima data dari server, dan banyak lagi. JavaScript juga mendukung pengembangan aplikasi web frontend dan backend.
JavaScript dapat digunakan pada semua platform, termasuk web, desktop, dan mobile. Saat ini, banyak kerangka kerja (framework) JavaScript yang populer seperti React, Angular, dan Vue yang digunakan untuk membangun aplikasi web frontend.
JavaScript juga dapat digunakan pada sisi server dengan Node.js, sehingga memungkinkan pengembang web untuk menggunakan bahasa yang sama baik pada sisi server maupun sisi klien. Hal ini membuat pengembangan aplikasi web menjadi lebih cepat dan efisien.
JavaScript dikembangkan oleh Brendan Eich pada tahun 1995 dan sekarang menjadi salah satu bahasa pemrograman web yang paling populer. Banyak perusahaan teknologi yang menggunakan JavaScript untuk pengembangan aplikasi web, seperti Google, Facebook, dan Twitter.
